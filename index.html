How to get depth data

Garmin Active Log GPX: ActiveCaptain → sync → export track GPX (active log, not saved). Drag here.

QuickDraw Community CSV: pip install qdc-converter
qdc-converter -i Contours -o depths.csv -l 1
Select the correct lake in TrollMap first, then drag the CSV. Points outside the lake bounds are dropped, depth meters → feet auto-converted, then gridded + contoured.

The contour engine is a JS port of qdc_to_trollmap.py – Marching Squares with NaN boundary clipping, so contours stay in the lake.

🗺️ QGIS Desktop Multi-Screenshot & XYZ Tile Integration Studio

If you georeference a batch of screenshots in desktop QGIS to build a seamless lake chart, you can bring them into TrollMap in three flawless ways:

1. GeoJSON Vector Tracing (Best Offline PWA Option)
In QGIS, extract the raster contours to vector lines and export as a .geojson file. Drag & drop that file into the Import Contours box above. It runs incredibly crisp at any zoom and stores permanently in browser memory forever.
